### Unstage Commits   
git rm --cached <file>   


### Start Tracking
git add <file>   
git add .   

   
### Commit   
git commit -m "message"   
git commit -am "message"

### Find diff   
git diff   
git diff -a   

### Merging   
(on main branch to receive merge) git merge <branch to take in>    
    
   
### Revert & Reset   
Revert - if commit is in remote repo   
Reset - if commit is in local repo
